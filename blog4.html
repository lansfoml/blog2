This week we will be exploring Javascript for the first time ever. I have some previous experience using CSS and HTML, but have never used or implemented Javascript into my work. HTML and CSS are markup languages that provide web pages structure and style. JavaScript is a programming language that allows you to construct interactive web pages. With javascript you can reveal or hide informtion just by clicking a button (which I will show below), change the color of items when they are hovered over, display animations, and more.

The code below shows how to make a button to display a date:

Go ahead, Click it!

<button type="button" onclick="document.getElementById('demo').innerHTML = Date()"> Click me to display Date and Time.</button> "

A big part of this is using getElementById(), to identify which attributes you would like to alter and what you would like to do to them. For example Javascript can also alter the CSS of particular attributes.

The code below is the code to alter the CSS of a particular attribute with the id of "example" when the button is pressed.

CLICK, CLICK, CLICK, CLICK

<button type="button" onclick="document.getElementById('example').style.fontSize='35px'">Click Me!</button>

JavaScript can change the style of an HTML element.

Javascript makes it easier to create responsive designs as I add more interaction and complexity to my programs, JavaScript has become an essential part of the Internet as a whole. It is necessary for search engines, ecommerce, content management systems, responsive design, social media, mobile apps, and more to exist. Making your site or app more interactive can benefit you and the user. Making the app or website easier to use, as well as more entertaining for you and the user, which also allows you to do more with your application.
