This week we will be learning about animations. If this works hopefully the text will be sliding into the screen right now. Transitions from one CSS style configuration to another can be animated using CSS animations. A style describing the CSS animation and a series of keyframes indicating the start and finish states of the animation's style, make up an animation. The code for the 2 animations will be listed below:

p {
 animation-duration: 3s;
 animation-name: slidein;
}
@keyframes slidein {
 from {
  margin-left: 100%;
  width: 300%;
}
  to {
  margin-left: 0%;
  width: 100%;
 }
}
p {
 animation-duration: 3s;
 animation-name: slidein;
}
@keyframes slidein {
 from {
  margin-left: 100%;
  width: 300%;
}
  to {
  margin-left: 50%;
  width: 100%;
 }
}

You can also make shapes and have animations for them. For example we made a div into a 50px x 50px square that slides in a sqaure while changing colors just by using simple css and animations. We have set it to run an infinite number of times, but you can set it to only do the animation however many times you want.

For the top paragraph the margin-left aspect in the "from" block starts at 100% so it starts all the way on the right. The margin-left in the "to" block then goes to 0% which means it will end up all the way on the left. For the paragraph right below the square the margin-left in the "to" block is switched to 50% so this paragraph only slide halfway across the screen. Animations are a cool and fun way to spice up your page as well as learn somehting new that will really catch the users eye.
