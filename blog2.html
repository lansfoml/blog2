For example this paragrpah is centered and has 25px of padding to the left and to the right. It can also make this sentence any color we want, including aquamarine or tomato.. Inline styles are defined in the anchor tag beginning with the word "style". More than one style can be changed in this inline " style=""" ". Inline styles can be good for styles that are not supposed to be shared with others. For example if we did not want every h1 to be green, then we would use this. If we wanted this header to have a border we could add " style="color: green; border-style:groove" " to give it a grooved border.
